# Filesystem

Some PalmSens devices, like the EmStat 4, EmStat Pico, Sensit Wearable, and Nexus) have an internal filesystem where they can store measurements (`.dmeas` files), and other data, e.g. generated by MethodSCRIPT. PyPalmSens provides a file-system-like interface to browse, read, write, and delete files on the connected instrument through [pypalmsens.CommProtocol][]. Asynchronous workflows are supported via [pypalmsens.CommProtocolAsync][].

Example:

```python
>>> import pypalmsens as ps
>>> instrument = ps.discover()[0]
>>> with ps.CommProtocol(instrument) as comm:
...     version = comm.query('i')
...     print(version)
[DevicePath('Measurements/17-07-2026/LSV-10-00-25-0.dmeas'),
 DevicePath('Measurements/17-07-2026/LSV-10-00-28-1.dmeas'),
 DevicePath('Measurements/16-07-2026/CV-14-46-50-2.dmeas')]
```

For more information how to use these classes, see [the documentation here](../communication.md).
